# _shared skills

Cross-agent operational runbooks. Any agent can load these.

- halt_protocol — kill switch tripped, positions at risk, or audit gap detected
- market_open_checklist — pre-open readiness run at 09:15 ET and 60 seconds before every 24/7 crypto session boundary
- black_swan_response — >3σ move on a primary index, flash crash, or exchange-wide outage
